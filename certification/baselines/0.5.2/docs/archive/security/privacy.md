# Privacy Policy

CODEC-CORTEX is a local-first protocol. It does not collect, transmit, or store personal data on remote servers.

- **No telemetry.** The CLI and codec do not phone home.
- **No cloud storage.** All `.cortex` and `.hcortex` files stay on your machine.
- **No analytics.** No usage data is collected.
- **No third-party sharing.** Zero data leaves your environment.

If you use PyPI (`pip install codec-cortex`), PyPI's own privacy policy applies to the download transaction. The installed package itself has no network calls.
