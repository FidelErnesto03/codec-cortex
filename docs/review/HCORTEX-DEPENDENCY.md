# Dependencia CE-7 — HCORTEX

F3-CHARTER exige que HCORTEX-CANONICAL reconstruya el AST original desde el canónico. El plan integral reserva la especificación y corpus HCORTEX para Fase 4.

Esta entrega garantiza únicamente las precondiciones de reversibilidad:

- cinco shapes explícitos;
- contratos y nombres de campo;
- foco;
- identidad local;
- orden significativo;
- glosario y extensiones preservados;
- bytes canónicos estables.

No se incluye un “HCORTEX mínimo” basado en encapsular bytes o JSON, porque eso aparentaría cumplimiento sin demostrar una representación humana canónica real.

**Estado CE-7:** `BLOCKED_PENDING_PHASE_4`.
