# Corpus HCORTEX 0.1

## Gate canónico

Cada caso `H001..H040` posee:

```text
CORTEX canónico
AST esperado
HCORTEX-CANONICAL
HCORTEX-READABLE
loss report CANONICAL
loss report READABLE
hashes y vector
```

## Inválidos

`invalid/` contiene documentos que deben fallar. `expected-diagnostics/` fija el código mínimo obligatorio.

## VIEW

`view-extension/` es un fixture no normativo excluido del Gate canónico.

El inventario procesable está en `manifest.json`.
